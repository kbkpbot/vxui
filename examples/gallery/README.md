# Gallery Demo

A comprehensive UI controls demonstration showcasing various desktop UI components built with vxui.

## Features

- **Buttons**: Primary, danger, and icon buttons with click handlers
- **Forms**: Text inputs, textareas, selects, checkboxes, radio buttons, sliders, and toggles
- **Progress Bars**: Animated progress indicators
- **Tabs**: Dynamic tab switching
- **Tables**: Data tables with add/delete operations
- **Modals**: Popup dialogs
- **Notifications**: Toast notifications with different types
- **Cards**: Flip cards animation

## How to Run

```sh
# From this directory
v run main.v
```

Or build and run:

```sh
v -prod -o gallery main.v
./gallery
```

## What It Demonstrates

This example shows how to:
- Handle various form input types
- Manage application state
- Implement real-time UI updates
- Create interactive components
- Use hx-swap-oob for partial page updates
